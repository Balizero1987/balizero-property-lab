# Instruksi untuk Claude

## Konteks Project
Ini adalah project **Property Lab** untuk Krisna - tools untuk section Real Estate di balizero.com

## Bahasa
- Semua output dalam **Bahasa Indonesia**
- Komentar kode dalam Bahasa Indonesia
- Dokumentasi dalam Bahasa Indonesia

## Tech Stack
- HTML5 + CSS3 (Tailwind via CDN)
- JavaScript vanilla atau Alpine.js
- Chart.js untuk grafik

## Design Guidelines
- Warna utama: #1e3a5f (biru), #c9a961 (gold)
- Font: Inter atau Poppins
- Mobile-first design
- Data akurat dan terpercaya

## Data Properti
- Harga dalam USD
- Area: Canggu, Ubud, Seminyak, Sanur, Uluwatu
- Rental yield: short-term vs long-term
- Opsi kepemilikan: Hak Pakai, Leasehold, PT PMA

## Prioritas
1. Kalkulasi harus akurat
2. Mobile responsive wajib
3. Angka realistis (berdasarkan market)
4. CTA untuk konsultasi
